# super-neon-snake
A colorful Neon-themed Snake Game built with HTML, CSS, and JavaScript. Features mobile-friendly touch controls and a high-score tracker for kids! 🍎🎮


# 🐍 Super Neon Snake Game

HTML, CSS aur JavaScript se bana ek mazedar aur colorful Snake Game!

## ✨ Features:
- 📱 Mobile-friendly touch controls
- 🏆 High score counter
- 🌈 Rainbow neon color-changing snake

## 🎮 Kaise Khele?
- **Mobile par:** Screen par diye gaye Up (▲), Down (▼), Left (◀), aur Right (▶) arrows ka use karein.
- **Computer par:** Keyboard ki Arrow Keys ka use karein.
